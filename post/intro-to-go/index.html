What is Go?

Go is an Open Source programming language. Go is typed, compiled and fast.

Go is Typed

Everything is Go has a type. User can define their own types too. What this lets us do is to have a language that is easy to read and hard to misuse.

Go is Compiled

Go source code compiles down to binary. To run a compiled Go application you do not need the Go runtime. So it Go is a good fit for containerized environment.

Go is Fast

When we consider programming language speed we often just think of the speed of the running program. And that is very important. I also think of developer productivity is also a huge factor. Because the fastest program would be written in assembly. We use higher level languages to make it easier to develop, understand and maintain. In terms of raw speed of execution Go is probably behind C/C++ or Rust. But in most cases is faster than Java, Python or Nodejs.
